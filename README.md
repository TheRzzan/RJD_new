# RJD_new

РЕАЛИЗАЦИЯ

Проект был реализован в среде разработки Android Studio с использованием языка Kotlin. 
Основа проекта - мультимодульная архитектура. Модули с пометкой api - выступают в роли интерфейсов для взаимодействия с фичей, а модули implementation - их реализацией. Объединяющим элементом выступает главный модуль app. База данных, применяемая в проекте - sqlite. 
Асинхронная работа осуществяется за счёт библиотеки rxJava2. В качестве архитктуры кода была использована MVVM.
В роли среды разработки дизайна выступало приложение Figma.


О ПРИЛОЖЕНИИ

Разработка программы велась в рамках тестового задания при трудоустройстве в компанию РЖД.
Согласно техническому заданию приложение должно было уметь хранить "контакты" двух категорий: друзья и коллеги. 
Функционал приложения: добавление, удаление и корректировка информации контактов, отображение списков конкретной категории.
